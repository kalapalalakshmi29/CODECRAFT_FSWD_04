# Setup Instructions

## Prerequisites
- Flutter SDK (3.8.1 or higher)
- Dart SDK
- Android Studio / VS Code
- Git

## Installation Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Harsha2675/CODECRAFT_FSWD_04.git
   cd CODECRAFT_FSWD_04/Project
   ```

2. **Install dependencies**:
   ```bash
   flutter pub get
   ```

3. **Run the app**:
   ```bash
   flutter run
   ```
